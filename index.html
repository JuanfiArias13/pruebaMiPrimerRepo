Divs y overflow

Para lograr este efecto de las cajas de colores utilizamos la etiqueta div, en HTML, para crear dos cajas y darles color con css, estos dos divs tienen la clase contenedor para darle estilos en comun en css como por ejemplo width, height, display:inline-block; (este para que dejen actuar como elementos de bloque y se nos permita tener dos en una misma linea, recordar que un elemento de bloque es aquel que toma toda la linea de nuestra p치gina sin importar su contenido y el elemento en linea toma solo el espacio en la pagina que tenga su contedio se ajusta) y con un id le damos aspectos mas "unicos" como por ejemplo el color, que como se podra apreciar son distintos en cada caja, no es el mismo color el que tenemos en "Icono de la p치gina" que el que tenemos en "Divs y overflow". Y alfinal usamos un tercer div para meter a los dos primeros y poder centrarlos y con la propiedad "text-align:left;" en el ".contenedor" dentro del css logramos que no se centre el texto de nuestras dos cajas de colores.
Y para lograr el efecto de scroll y que el contenido de las cajas no se salgan utilizamos la propiedad "overflow:scroll;" en el css, esta es una de las propiedades que ira en el ".contenedor" para que no tenga que repetirse en el codigo y se aplique a ambas cajas.
